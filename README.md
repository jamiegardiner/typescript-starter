# Typescript Starter

This is a starter project for Typescript. It includes:

- typescript
- pnpm
- vitest
- biome
- husky
- lint-staged
- commitlint with Conventional Commits

Install dependencies:

```bash
pnpm install
```

Run tests:

```bash
pnpm test
```

Run tests with coverage:

```bash
pnpm test:coverage
```

Run format:

```bash
pnpm format:check
pnpm format
```

lint-staged and husky will run biome on staged files before commit.
